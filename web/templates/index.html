Available Algorithms

Monte Carlo Tree Search (MCTS)

A tree search algorithm that uses random sampling to evaluate moves and find optimal decisions. It balances exploration and exploitation using UCT (Upper Confidence Bound for Trees).

Run MCTS

Rapidly-exploring Random Tree (RRT)

A randomized algorithm designed to efficiently search high-dimensional spaces by building a space-filling tree. It grows by randomly sampling points and connecting them to the nearest node.

Run RRT
